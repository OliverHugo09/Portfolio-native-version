Openings Cloud

Openings Cloud es una aplicación web basada en React que ofrece una agradable experiencia para visualizar openings a través de diversos servicios en la nube, como Mega y otros proveedores de almacenamiento. Permite a los usuarios acceder y gestionar fácilmente sus openings desde distintas plataformas en un solo lugar, brindando comodidad y accesibilidad a su contenido almacenado en la nube.

Chat Ionic

Aplicación de mensajería que combina Angular con Ionic, Node.js y Express. Ofrece una experiencia completa de chat con autenticación segura mediante Web Tokens. Permite a los usuarios subir imágenes de perfil utilizando Cloudinary y emplea Web Sockets para una comunicación en tiempo real. Además, integra diversas APIs para maximizar su funcionalidad, lo cual ofrece una experiencia completa al usuario.

MVP Sistema de Recursos Humanos

Este sistema, construido en Angular con Ionic, ofrece autenticación de usuarios, calendario de eventos, solicitud de asesorías, blogs compartidos y un registro detallado de empresas y usuarios. Está diseñado para gestionar eficientemente interacciones y comunicaciones, proporcionando un entorno completo para una gestión efectiva de recursos humanos.
